Plane Details Viewer

Your task is to create simple plane details viewer, that allows the user to view some of the basic information. It's laboratory task only (12pts). If any doubts according to task, please follow the sample application or ask your teacher.

Laboratory Part

  • Sample Application
  • Layout:
    • The application startup location is set to screen center.
    • Application's size is set and fixed at 600x350.
    • Application is divided into two areas.
      • Left one containing list of planes.
      • Right one containing details about selected plane.
  • Input Data:
    • Data is given as a list inside your MainWindow.xaml.cs class.
    • Required for the solution classes are located in Plane.cs file.
  • Binding:
    • Use Binding for all elements and UI changes.
    • In C# you can only implement Converters.
    • No code-behind in C# is allowed. Except that which loads data.
  • Converters:
    • Use converter to display weight of the plane in kilograms instead of tones.
    • Use converter to compute arrival time based on departure time and flight duration.
  • Styles:
    • Your application should use the same styling as sample application.
      • Use Bold and Italic font for Planes and Flights Lists and Airports.
      • Use Vertical and Horizontal text alignment for every text block in the application.
    • Define at least one style to avoid repeating styling code.
  • Hints:
    • GroupBox, ListView, TextBlock, IValueConverter, IMultiValueConverter, MultiBinding, DataContext with ElementName, StaticResource, ItemsPanelTemplate, DataTemplate.
  • Points:
    • User Interface:
      • Left panel with list - 1p
      • Right panel with details - 2p
      • List horizontal content alignment - 1p
    • Styles:
      • Styles definition and usage - 1p
    • Binding:
      • C# defined data binding - 1p
      • Plane details binding - 1.5p
      • Flight details binding - 1.5p
    • Converters:
      • Weight Converter C# - 1p
      • Arrival Time Converter C# - 1p
      • Converters' XAML - 1p
